# Savitri Balika Admin Dashboard

Modern React dashboard for Savitri Balika Inter College admin use.

## Features

- Live school header with logo, date, and time
- Welcome dashboard hero section with school building images
- Main school website link
- Class-wise admission links
- Book list filter by class and session
- School brochure link
- Voice note link
- Required documents section
- Facebook poster link

## Live Links Used

- School Website: `https://savitribalikaintercollege.netlify.app/`
- Admission Form `Nursery to 5th`: `https://bal-vidya-mandir.vercel.app/`
- Admission Form `6th to 8th`: `https://school-form-j-h.vercel.app/`
- Admission Form `9th to 12th`: `https://school-admission-form-delta.vercel.app/`
- School Brochure: `https://savitribalikaintercollege.in/`
- Voice Note: `https://notice-audio-er76.vercel.app/`

## Project Structure

- `src/App.jsx` main dashboard layout
- `src/index.css` dashboard styling
- `src/data/dashboardData.js` links and dashboard content
- `public/booklists/` uploaded class book list HTML files
- `public/documents/` admission document files

## Run Locally

```bash
npm install
npm run dev
```
